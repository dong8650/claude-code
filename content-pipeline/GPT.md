[GPT Content Generation System v2]

목표:
“잘 쓴 대본”이 아니라
“조회수 1만 이상 가능성 있는 대본 생성”

--------------------------------
1. 콘텐츠 타입 선택 (필수)
--------------------------------

아래 중 하나 선택하여 생성

- emotion (독설)
- ranking (순위)
- money (계산)
- quote (명언)
- hybrid (혼합)

--------------------------------
2. 공통 구조
--------------------------------

Hook (8자 이하)
- 즉시 이해 + 충격
- 정체성 단정 / 폭로 / 선택 강요

Body (3~5문장)
- 설명 금지
- 현실 문장 1개
- 반전 문장 1개

Closing (10자 이하)
- Hook 재해석
- 낙인 / 비꼼 / 역설

--------------------------------
3. 반복 시청 설계
--------------------------------

- closing이 hook 의미 뒤집어야 함
- 다시 듣고 싶게 만들 것

--------------------------------
4. 조회수 기준 (핵심)
--------------------------------

반드시 아래 느낌 충족

- 기분 나쁘지만 맞는 말
- 댓글 달고 싶어짐
- 나한테 하는 말 같음

--------------------------------
5. 콘텐츠 타입별 규칙
--------------------------------

[emotion]

- "너 / 넌 / 니" 포함
- 정체성 공격
- 클리셰 금지

--------------------------------

[ranking]

- TOP3 / TOP5 형태
- 숫자 필수
- 충격 요소 포함

--------------------------------

[money]

- 숫자 + 기간 필수
- 현실 계산 기반
- 마지막 감정 압박

--------------------------------

[quote]

- 일반 명언 금지
- 현실 비틀기 필수

--------------------------------

[hybrid]

- 2개 이상 타입 결합
- 예: money+emotion

--------------------------------
6. 데이터형 생성 규칙
--------------------------------

data형 주제는 아래에서만 생성

- 노후 계산
- 월급 현실
- 지출 비교
- 인생 후회
- 순위
- 체크리스트

주제 공식:

[숫자] + [현실 상황] + [불안/시간]

예:

- 3억이면 몇 년 버티냐
- 월급 300이면 남는 돈
- 40대 후회 TOP5

--------------------------------
7. 금지 규칙
--------------------------------

- 설명형 문장
- 추상 표현
- 일반 상식
- 너무 넓은 주제

--------------------------------
8. 출력 형식
--------------------------------

{
  "content_type": "",
  "hook": "",
  "script_ko": "",
  "closing_ko": "",
  "pattern_type": ""
}
