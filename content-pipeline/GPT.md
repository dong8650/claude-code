# GPT 콘텐츠 생성 프롬프트 & 전략 가이드

## 역할
- 아이디어, 대본, 콘셉트, 감정 후킹 등 “머리 역할”에 집중

## 목표
- 100만 조회수 쇼츠 자동 생성
- 10개 중 1개 이상 ‘터지는’ 구조 도달

---

## 작동 규칙

- 첫 문장 8자 이하 임팩트(직격 Hook)
- 전체 3~5문장, 총 60~100자
- 설명형/무감정/긴 문장 일체 금지
- 감정, 반전, 임팩트, 반복 시청/저장/공유 유도 포함
- 영상 길이 18~25초

---

## 스크립트 생성 프롬프트(템플릿)

```
내일 이 쇼츠를 100만 명이 본다고 가정하고, 아래 구조를 반드시 지켜서 인간처럼, 캐치프레이즈와 액션을 섞어 대본을 써라.

1. Hook(직격): 8자 이내, 타깃(30~40대 직장인) 심리 자극 (분노·불안·공포·질투)
   (예: '월급은 덫이다', '지금 늦었어요')
2. 공감: 현실 고민, 욕망, 불안 등 진짜 타깃 문제
3. 뒤집기: 상식 or 흔한 조언 뒤집기, 반전, 불편한 진실
4. 행동(저장/공유/재시청): 10자 이내, 강제 자극 문구

- 설명문·평서문·긴 문장 금지
- 각 문장 scroll_stop_power, emotional_attack ≥ 7, repeat_value ≥ 6로 평가
- 반드시 3~5문장·60~100자 이내

예시 대본:
"퇴사하세요.
마흔 살에도 불안하죠.
저축만으론 절대 못 부자 돼요.
이건 저장각."
```

---

## 추가 운영 지침

- Hook/공감/뒤집기/행동 등 각 유형별 문구 리스트 형태로 프롬프트에 랜덤 또는 조건 삽입
- 설명형/감정어 결여된 문장 자동 필터링 로직 유지
- 각 문장별 scroll_stop_power, emotional_attack, repeat_value 자동 평가/로그화
