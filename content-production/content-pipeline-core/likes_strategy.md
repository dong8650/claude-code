# 좋아요 설계 전략 — 매일의 설계 / 건강편

> 작성: 2026-05-05  
> 목표: 좋아요율 0.25% → 5%+ 달성을 위한 3가지 코드 수준 변경

---

## 현황 진단

| 지표 | 현재 | 목표 |
|------|------|------|
| 좋아요율 | ~0.25% | 5%+ |
| 완시율 | 49.1% (mindset) | 85%+ |
| 채널 평균 | B급 (2~3k 정체) | S급 (10만+) |

**근본 원인**: 영상이 "오, 그렇구나"로 끝남 → 시청자가 행동(좋아요·저장·공유)을 안 함

---

## 시안 ① — CTA 오버레이 (마지막 1.2초)

### Before
```
[영상 끝]  ← 아무 행동 유도 없음
```

### After
```
[영상 끝 1.2초 전부터 표시]
━━━━━━━━━━━━━━━━━━━━━━
  공감됐으면 좋아요  저장해두세요
━━━━━━━━━━━━━━━━━━━━━━
```

### 적용 스펙

| 항목 | 값 |
|------|-----|
| 텍스트 | `공감됐으면 좋아요  저장해두세요` |
| 폰트 크기 | 36px |
| 색상 | `#FFD700` (골드) |
| 테두리 | 2px black@0.8 |
| 위치 | 하단 바 위 50px (`1920 - bot_bar_h - 50`) |
| 표시 시점 | 마지막 1.2초 |

**enable 조건**
- content-health: `enable='gte(t,{voice_dur-1.2:.1f})'`
- content-mindset: `enable='between(t,{N_main-1.2:.1f},{N_main:.1f})'`

### 왜 효과적인가
- 영상 끝날 때 좋아요 버튼이 화면에 있음 → 텍스트와 버튼이 같은 시야에 들어옴
- 1.2초는 손가락이 반응할 수 있는 최소 시간
- "공감됐으면"이라는 조건부 문장 → 거부감 없이 자발적 행동 유도

## 시안 ② — Closing 4패턴 규칙 (generate_script.py)

### Before
```
CLOSING: 최대 15자, 관점 전환 한 문장
→ "말할 때 비로소 산다"  ← 시청자가 "오, 그렇구나"로 끝냄
```

### After
```
CLOSING: 최대 20자, 좋아요·저장·공유 유발 — 4패턴 중 1개
```

### 4가지 Closing 패턴

| 패턴 | 효과 | 예시 |
|------|------|------|
| ① 공감형 | 공감+안도 → 좋아요 | "나만 이랬던 거 아니었어" |
| ② 저장형 | 손실 공포 → 저장 | "이거 저장 안 하면 또 당한다" |
| ③ 승리감형 | 독점감 → 좋아요+공유 | "이 영상 본 사람만 안다" |
| ④ 공유형 | 사회적 책임감 → 공유 | "지금 주변에 퍼뜨려야 함" |

### 패턴 선택 가이드

| content_type | 권장 패턴 |
|---|---|
| emotion | ① 공감형 (공감이 이미 폭발 상태) |
| ranking | ③ 승리감형 (정보 독점감 극대화) |
| money | ② 저장형 (손실 공포 → 저장) |
| quote | ④ 공유형 (통념 뒤집기 → 퍼뜨리기) |

### 왜 기존 "관점 전환"이 좋아요를 못 받는가
- "말할 때 비로소 산다" → 지식적 깨달음 → **좋아요 행동 없음**
- "나만 이랬던 거 아니었어" → 감정적 공감 → **좋아요 누름**

## 시안 ③ — Scene6 좋아요+저장 동시 유도 (generate_script_v2.py)

### Before
```
scene6: 저장유도 (duration ~2): "저장해두고 내일부터 시작해 💾" — 구체적 행동 촉구
```

### After
```
scene6: 좋아요+저장유도 (duration ~2): "공감됐으면 좋아요 누르고 저장해둬 💾👍" — 좋아요+저장 동시 촉구
```

### 자막 스타일 변경
- 기존: `Save` 스타일 (노란색) — 저장만
- 변경: 좋아요·저장 동시 → 기존 `Save` 스타일 유지 (충분히 눈에 띔)

### 왜 좋아요를 저장 앞에 언급하는가
- 좋아요는 1초짜리 행동 (탭 한 번)
- 저장보다 마찰이 낮음 → 먼저 제안해서 행동 시작 유도
- 저장 언급으로 "가치 있는 콘텐츠"라는 인식 강화
